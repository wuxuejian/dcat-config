![](https://ghost-ai.com/images/original/file/1617172863498956751.png "")

### composer 安装

> composer require ghost/dcat-config

### dcat config 配置

> admin_setting 添加 ghost::admin_config 字段

> key=>value形式保存

> 可以替换系统config 或env 配置
> 使用 config("key");

### 国际化 

```php
//resources/lang/zh-CN/dcat-config

return [
    'Basic'=>'基本配置'
];

```
